Selling Property

firebase의 authentication, firestore, storage 기능을 활용해 만들어진 부동산 웹앱입니다. 로그인, 회원가입 및 새로운 매물을 올리고 수정하고 삭제할 수 있습니다. 오류가 발생할 시 toastify를 통해 에러 메세지를 전달하고 react leaflet을 사용하여 지도를 통해 매물의 위치를 확인할 수 있게 만들었습니다.

  • Firebase
  • Auth
  • Firestore
  • Storage
  • Tostify
  • React leaflet

Github Finder

깃허브 REST API를 통해 사용자를 검색하고 프로필 정보와 respository를 보여주는 웹앱입니다. 입력값이 없는 상태로 검색을 할 시 에러 메세지가 나타났다 사라집니다. React hooks (useContext, useReducer, useEffect)와 tailwind css의 daisy UI 컴포넌트를 사용했습니다.

  • Github API
  • React
  • Javascript
  • tailwind CSS
  • Vercel

Feedback App

피드백을 작성하고, 삭제하고, 수정할 수 있는 웹앱입니다. React hooks와 Json-server를 활용하였고 피드백은 최소 10개 이상의 문자를 입력해야 작성됩니다.

  • React
  • Json-server

  • Netlify

Tomorrow House

Tomorrow House는 모든 화면에 적용 가능한 반응형 웹 사이트 입니다. Javascript와 scss를 활용해 만들었고 오늘의 집 사이트를 클론코딩한 사이트입니다.

  • JavaScript
  • SCSS

Momentum App

Momentum App은 사용자의 집중력 향상을 위해 만들어진 웹앱입니다. 사용자의 위치를 통해 날씨를 보여주고, 사용자가 입력한 데이터들은 localstorage에 저장이됩니다. todo 버튼을 클릭해 할 일 목록을 작성하고 삭제할 수 있습니다.

  • JavaScript
  • CSS
  • Localstorage

Paint JS

Paint Js는 간단한 그림이나 메모를 남길 수 있는 웹앱입니다. canvas와 javascript를 사용해 만들어졌고, 색상 선택과 선 두께 조절이 가능하며 save 버튼을 눌러 그림을 저장할 수 있습니다.
